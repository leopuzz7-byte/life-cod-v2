## Стиль Надежды (обязательные правила для промтов)

```
ЗАПРЕЩЕНО:
- Длинное тире (—). Только запятая или короткий дефис (-)
- «Год трансформации», «год роста», «год возможностей» — общие фразы
- Повторять одну мысль разными словами
- Эзотерические рассуждения без конкретики
- Справочник по аркану вместо персонального прогноза

ОБЯЗАТЕЛЬНО:
- Сначала факт/расчёт, потом трактовка, потом практический вывод
- Конкретные рекомендации: не «занимайтесь спортом», а «плавание, ходьба, велосипед»
- Человек должен видеть свою жизнь, а не описание архетипа
- События важнее философии
- Язык «вы/ваш», тепло, без напыщенности
- Каждый блок отвечает на свой вопрос, а не повторяет другой
```

## Структура по ТЗ Надежды

### ГОДОВОЙ ПРОГНОЗ (БЛОК 0-13)

| Блок | Название | Что показывать |
|---|---|---|
| 0 | Расчёт | Аркан года рождения + Аркан интересующего года = Итог. Показать математику |
| 1 | Главный аркан | Номер, название, планета, стихия + полная трактовка энергии |
| 2 | Введение в ваш год | Почему этот аркан пришёл именно сейчас, главные темы |
| 3 | Атмосфера года | Общие тенденции, что усилится, что уйдёт |
| 4 | Деньги и работа | Возможности, риски, ошибки, способы заработка, стоит ли менять работу |
| 5 | Отношения | Отдельно: одинокие / в отношениях. Плюсы, минусы, риски |
| 6 | Здоровье | Чувствительные зоны, что вредно, что полезно, конкретные практики |
| 7 | Главные возможности | Что открывается, куда смотреть |
| 8 | Главные риски | Ошибки, ловушки, что нельзя делать |
| 9 | Плюсовое проявление | Как проживать правильно, что усиливает удачу |
| 10 | Минусовое проявление | Деструктивные сценарии, как распознать |
| 11 | Рекомендации | Только конкретика: что делать / чего не делать |
| 12 | Итог года | Для чего дан этот год, главный урок |
| 13 | По месяцам (VIP) | Отдельный треугольник для каждого месяца |

**Расчёт (БЛОК 0):**
- Аркан года рождения = 4-я позиция личной матрицы
- Аркан года = по системе (из кода уже вычисляется)
- Итог = сумма с редукцией
- Показать клиенту: `Аркан года рождения: 3 Императрица + Аркан 2027 года: 11 Сила = 14 Умеренность`

---

### МЕСЯЧНЫЙ ПРОГНОЗ

**Расчёт треугольника:**
- Позиция 2 (аркан месяца): `2-я поз. человека + номер месяца`
- Позиция 4 (аркан года рождения): `2-я поз. + аркан года`
- Итог (главный аркан месяца): сумма с редукцией

**Секции:**
1. Расчёт (показать математику, как у Надежды)
2. Треугольник — три квадратика с арканами *(уже есть)*
3. Главная энергия месяца (название, планета, ключевая тема)
4. Общий прогноз месяца
5. Глубокое влияние аркана 1 (фоновая энергия)
6. Глубокое влияние аркана 2 (энергия самого месяца)
7. Как проявляется в жизни
8. Деньги и работа
9. Отношения
10. Здоровье
11. Главный риск месяца
12. Главные возможности
13. Практические рекомендации

**Пример расчёта (от Надежды, МАРТ 2027, д.р. 07.07.YYYY):**
```
2 позиция: 7 + 3 (март) = 10
4 позиция: 3 + 11 (аркан 2027) = 14
Итог: 10 + 14 = 24 → 2

Треугольник: 10 Колесо Фортуны — 14 Умеренность — 2 Жрица
```

---

### ДНЕВНОЙ ПРОГНОЗ (13 позиций матрицы)

**Позиции матрицы дня:**
1. Главная энергия дня
2. Утро / старт дня
3. Потенциал дня
4. Самооценка
5. Цель дня
6. Главный ресурс дня
7. Основа дня (фундамент)
8. Главная задача дня
9. Способ решения задач
10. Итог дня
11. Подсознательные процессы
12. Внешнее влияние
13. Кармическая задача дня

**После 13 позиций:**
- Глубокий энергетический разбор (связь всех позиций)
- Стратегия дня (утро / день / вечер) — три отдельных блока
- Деньги, карьера, отношения, здоровье
- Риски дня
- Возможности дня
- Практика дня (конкретное упражнение)
- Итог и главный вывод

## Важные технические детали

- **Файлы >150 строк**: только Python str.replace() через bash
- **После каждого replace**: `tail -3 файл && wc -l файл`
- **Кеш**: при изменении промта — поднять версию (`ai_year_v1_` → `ai_year_v2_`)
- **Проп-дриллинг**: при добавлении `aiLoading` в новый компонент — проверить всю цепочку
- **Git push**: давать команду для cmd, не пушить из sandbox
