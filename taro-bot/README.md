# Телеграм-бот Надежды

Воронка знакомства и разбор Предназначения по методике 1, с подпиской на канал и чатом с ИИ-тарологом.

## Что делает бот

1. `/start`: приветствие Надежды (место под видео-кружок, пока текст по скрипту).
2. Спрашивает имя, потом дату рождения (ДД.ММ.ГГГГ).
3. Анимация ожидания (Считаю, Смотрю на карты, Ещё немного).
4. Три карты жизненных периодов на бело-кремовом фоне (до 25, 25-50, после 50) плюс короткий мини-разбор.
5. Барнум-фраза про день недели и сезон рождения, крючок на подписку.
6. Проверка подписки на канал, после подписки подробный разбор и картинка полной матрицы из 12 позиций.
7. Меню: разбор дня (бесплатно 1 раз в день), разбор месяца, совместимость, расклад по вопросу, чат с ИИ-тарологом (3 сообщения в день), калькулятор (мини-апп), канал и консультация.
8. Заглушка оплаты Robokassa, реферальная программа (пригласи друга, оба получают плюс сообщение ИИ).

## Как запустить у себя

Нужен Node.js 18+.

```bash
cd taro-bot
npm install
cp .env.example .env   # если .env ещё нет
# впиши в .env токен бота, канал, ключ ИИ, URL калькулятора
npm start
```

Бот работает на long-polling, держится пока запущен процесс. Для постоянной работы поставь на сервер (pm2, systemd или любой хостинг Node).

## Переменные .env

- `BOT_TOKEN` токен от BotFather
- `CHANNEL` юзернейм канала для проверки подписки, например @taro_lifecod (бот должен быть админом канала)
- `CHANNEL_URL` ссылка на канал
- `CALC_URL` адрес калькулятора для мини-аппа
- `AI_API_KEY`, `AI_API_URL`, `AI_MODEL` доступ к ИИ (proxyapi, gpt-4o-mini)
- `ROBOKASSA_*` пусто, впишем при подключении оплаты

Файл `.env` в `.gitignore`, в репозиторий не попадает.

## Рассылки (cron)

Отправка требует запущенного бота и планировщика. Примеры для cron на сервере:

```bash
# ежедневный вопрос неплативших в 11:00
0 11 * * *  cd /path/taro-bot && node src/bot/run-broadcast.js daily
# еженедельный разбор по понедельникам в 10:00
0 10 * * 1  cd /path/taro-bot && node src/bot/run-broadcast.js weekly
```

Тексты вопросов в `src/bot/broadcasts.js`, черновые, финальные лучше от Надежды.

## Что пока заглушка

- Видео-кружок на старте (пока текст, вставим когда Надежда запишет).
- Оплата Robokassa (кнопка есть, живую оплату подключим, место под ключи в .env готово).
- Разбор месяца и расклад по вопросу (заглушки с предложением подписки).

## Структура

```
taro-bot/
  .env                секреты (не в git)
  src/
    bot/
      index.js        весь сценарий и меню
      config.js       конфиг из .env
      store.js        файловое хранилище пользователей и лимитов
      broadcasts.js   банк вопросов, рассылки, рефералка
      run-broadcast.js запуск рассылки из cron
    engine/           расчёт матрицы и данные арканов (копия из калькулятора)
    render/           картинки: три карты и полная матрица (canvas)
    ai/reading.js     все ИИ-генерации, тон на «ты», без длинных тире
  assets/arcana/      22 карты арканов
  data/users.json     база пользователей (создаётся при запуске, не в git)
```

Калькулятор (папка `src/` проекта life-cod-v2) не затрагивается, движок скопирован отдельно.
